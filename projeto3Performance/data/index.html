Grupo de Máquina — ISO 10816-3
Como identificar seu grupo: o grupo depende da potência e do tipo de fundação da máquina, não apenas do tamanho. Referência: ISO 10816-3 ↗
Grupo I
Máquinas pequenas
até ~15 kW
A < 0,71 B até 1,80 C até 4,50 D > 4,50
mm/s RMS
Grupo II
Máquinas médias
15–75 kW, base rígida
A < 1,12 B até 2,80 C até 7,10 D > 7,10
mm/s RMS
Grupo III
Máquinas grandes
>75 kW, fundação rígida
A < 1,80 B até 4,50 C até 11,20 D > 11,20
mm/s RMS
Grupo IV
Máquinas grandes
>75 kW, fundação flexível
A < 2,80 B até 7,10 C até 18,00 D > 18,00
mm/s RMS
⚠️ Na dúvida entre Grupo III e IV, o critério decisivo é a rigidez da fundação, não a potência da máquina.
Temperatura
⚠️ Não existe uma classificação universal de temperatura equivalente à ISO 10816 para vibração. Os limites de temperatura variam conforme o fabricante, tipo de rolamento, lubrificante utilizado e condições de operação específicas de cada máquina. Consulte o manual do equipamento ou o fabricante para obter os limites adequados.
Wi-Fi
Frequência de amostragem
A frequência de amostragem define a faixa de vibração que o sensor consegue capturar. Pelo teorema de Nyquist, a faixa capturada é metade da frequência configurada — ex: 800 Hz de amostragem captura até 400 Hz de vibração. Frequências mais altas aumentam o consumo de CPU e memória.

VITRA

Vibração e Temperatura como Rastreamento de Anomalias

Sistema embarcado de monitoramento de vibração e temperatura conforme ISO 10816-3, desenvolvido com ESP32.

Universidade

Pontifícia Universidade católica do Paraná

Integrantes

  • Arlen abtibol, Lucas Koiti, Luis Lustosa, Luiz Dias, Luiz Simm, Rafael Sato

LEDs indicadores

LED de vibração — zonas ISO 10816 conforme grupo configurado:

  • 🟢 Verde — Zona A (ótimo)
  • 🟡 Amarelo — Zona B (aceitável)
  • 🟠 Laranja — Zona C (atenção)
  • 🔴 Vermelho — Zona D (crítico)

LED de temperatura — escala de referência geral para maquinário:

  • 🔵 Azul — abaixo de 40°C (frio / ambiente)
  • 🟢 Verde — 40–70°C (operação normal)
  • 🟡 Amarelo — 70–90°C (atenção)
  • 🔴 Vermelho — acima de 90°C (crítico)

⚠️ Os limites de temperatura são referência geral. Consulte o fabricante da máquina para limites específicos do seu equipamento.

Changelog
v1.0.0 — Base do sistema
Abril 2026
  • Firmware ESP32 com FreeRTOS: 3 tasks (Comunicacao, Sensores, Armazenamento)
  • Interrupção de hardware via timer para coleta do ADXL345
  • Leitura de temperatura via DS18B20 (assíncrona, sem bloqueio)
  • Servidor web embarcado com LittleFS
  • Configuração de Wi-Fi via interface sem regravação de firmware
  • Interface web inicial com gráficos de tempo real
v1.1.0 — Conformidade ISO 10816
Abril–Maio 2026
  • Cálculo de vibração por integração espectral da FFT (mm/s RMS)
  • Remoção de gravidade com filtro passa-alta exponencial
  • Frequência de amostragem configurável via interface (10–2000 Hz)
  • Buffer de amostras ampliado para 256 pontos
  • Persistência de frequência configurada na flash
v1.2.0 — Histórico e CSV
Maio 2026
  • Histórico de 24h: médias de 5 em 5 minutos (288 registros)
  • Persistência do histórico na flash com carregamento no boot
  • Exportação CSV com estatísticas e coluna de grupo ISO
  • 4 gráficos: tempo real (5 min) e histórico (24h)
  • Gráficos históricos persistem após reinício e atualização de página
v1.3.0 — Interface e Performance
Maio 2026
  • Interface reestruturada: menu lateral, 5 abas
  • Aba Performance: heap, uptime, Wi-Fi, RSSI, uso de CPU, tempo de execução de funções
  • Aba Logs: terminal ao vivo com níveis INFO/WARN/ERRO
  • Responsividade mobile com menu hamburguer
  • Gráficos com escala fixa 0–20 mm/s, eixos e legenda
v1.4.0 — Classificação ISO 10816-3
Maio–Junho 2026
  • Seletor de grupo de máquina (I, II, III, IV)
  • Faixas de cor nos gráficos por zona ISO (A, B, C, D)
  • Marcação vertical ao trocar de grupo com cores por período
  • Badge de zona atual no header
  • LED de vibração com zonas ISO dinâmicas
  • LED de temperatura com escala industrial (azul/verde/amarelo/vermelho)
  • Coluna de grupo ISO no CSV
v1.5.0 — Hardware e Energia
Junho 2026
  • Suporte a LCD 16x2 I2C com IP e modo Wi-Fi
  • 2 LEDs RGB: vibração (zonas ISO) e temperatura (escala industrial)
  • Modo econômico automático após 30s sem acesso
  • Polling adaptativo: intervalos aumentam com inatividade do usuário
  • Wi-Fi modem sleep ativo em operação normal
